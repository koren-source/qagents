# agentmat

Koren's content agent stack.

## Agents

### Matt (Content Planner) 🟡 Dormant
Full SELL-framework content planner. Mines ideas from Notion Idea Vault, memory logs, and YouTube → scores via Sonnet ($10K + Hormozi filters) → writes full briefs via Opus → publishes to Notion + Slack.

**Activation:** Set `DORMANT_MODE = false` in `matt/index.js`

See `matt/README.md` for full docs.
